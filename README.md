# sql-server-practice-db
Hands-on SQL Server project to practice database fundamentals and SQL queries.
This project simulates a simple database monitoring environment with a customer, database servers, databases, monitoring checks, and incidents.

## What This Project Includes
- Creating tables
- Inserting data
- Using relationships between tables
- Primary keys and foreign keys
- Basic queries (SELECT, JOIN, LEFT JOIN, WHERE, ORDER BY, GROUP BY)

## Example Queries
- Show databases with active issues
- Show unresolved incidents
- Count issues per server
- Show all databases, including databases without monitoring checks

## Purpose
I built this project to gain hands-on SQL experience and better understand how databases work in practice.

## Tools Used
- SQL Server Management Studio (SSMS)
